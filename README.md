# Ficheros

Este repositorio contiene ejercicios y prácticas en **C** enfocados en el manejo de **ficheros**. El objetivo principal ha sido aprender a trabajar con archivos, lectura y escritura de datos, así como manejar estructuras relacionadas con ficheros en C.

## Descripción

El código incluido en este proyecto muestra distintos ejemplos para abrir, leer, escribir y cerrar ficheros usando funciones estándar de C. Estos ejercicios ayudan a comprender cómo se gestiona la entrada y salida de datos en programas que requieren persistencia en archivos.

## Tecnologías utilizadas

- Lenguaje **C**
- Compilador compatible con C (por ejemplo `gcc`)
